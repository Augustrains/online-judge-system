竞赛介绍

本次计算机竞赛旨在为广大计算机爱好者提供一个展示自我、交流学习的平台。竞赛涵盖多个领域，包括算法设计、软件开发、人工智能等。无论你是初学者还是经验丰富的开发者，都能在本次竞赛中找到属于自己的挑战。

参赛规则

  • 参赛对象为在校大学生及相关专业从业者。
  • 参赛作品必须为原创，不得抄袭他人成果。
  • 每个参赛团队人数不超过 5 人。
  • 参赛作品需在规定时间内提交。

一等奖

奖金 10000 元 + 荣誉证书

二等奖

三等奖
